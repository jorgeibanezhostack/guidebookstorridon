Laundry Service SOP

✓ 0/17 tasks ⏰ All day job
⏰ Timing: This is an all-day job with waiting periods. Start early, run multiple loads throughout the day. Machines in laundry room off main hallway.
🧺 Step 1
Sorting & preparation
▼
Collect all dirty laundry from B&B rooms and cottages
Sort into piles: whites, colors, towels, bedding
Check all items for stains or damage
Pre-treat any visible stains with stain remover
🔍 Sorting guide: Whites (sheets, pillowcases) separate from colors. Towels together (heavy load). Bedding separate (duvet covers, blankets). Never mix guest and staff linens.
🌀 Step 2
Washing cycles
▼
Load first batch (don't overfill - leave space for water)
Add detergent (follow machine instructions for amount)
Select appropriate cycle: hot for whites/towels, warm for colors
Start machine and note completion time
Run additional loads throughout day (rotate batches)
⚠️ Machine care: Don't overload machines. Remove items promptly after cycle ends to prevent mildew. Leave door slightly open between loads for ventilation.
☀️ Step 3
Drying process
▼
Transfer washed items to dryer immediately after wash cycle
Clean lint filter before starting dryer
Set dryer to appropriate heat: high for towels, medium for everything else
Run dryer cycle and check if fully dry (may need extra time)
Remove items promptly to minimize wrinkles
💡 Drying tip: Shake out items before putting in dryer to reduce drying time. Don't over-dry delicate items. Check pockets for forgotten items.
📦 Step 4
Folding & storage
▼
Fold all sheets and pillowcases neatly
Fold all towels in thirds (standard hotel fold)
Fold duvet covers and blankets
Separate guest linens from staff linens
Store guest linens in upstairs cupboard (brown stairs, first door right)
Store staff/volunteer linens in laundry room cupboard
Return any cottage linens to cottage storage cupboards
Leave laundry room clean and tidy for next use
🗄️ Storage locations: Guest linens: Upstairs linen cupboard (brown stairs, first door right). Staff linens: Laundry room cupboard. Cottage linens: Return to each cottage storage.
